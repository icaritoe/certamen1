# 🌐 Portafolio Web - Angela Melgarejo Letelier

Este proyecto es un **portafolio personal** desarrollado con **HTML5 y CSS3**, que muestra información sobre mí, mis habilidades, proyectos realizados y formas de contacto.  
El objetivo es practicar **maquetación responsiva**, aplicar estilos modernos y crear una experiencia accesible y clara.

---

## 📂 Estructura del proyecto

- **index.html** → Contiene la estructura principal del sitio:
  - **Header** con logo y barra de navegación.
  - **Sección "Sobre mí"** con presentación personal y datos básicos.
  - **Sección "Habilidades"** con tabla de tecnologías y barras de progreso.
  - **Sección "Proyectos"** con tarjetas de proyectos y enlaces a repositorios.
  - **Sección "Contacto"** con formulario y datos de contacto.
  - **Footer** con logo y enlaces a redes.

- **style.css** → Contiene los estilos del sitio:
  - Reset básico de márgenes y paddings.
  - Tipografía **Roboto** desde Google Fonts.
  - Colores principales: azul oscuro, blanco y tonos grises.
  - Uso de **Flexbox y Grid** para maquetación responsiva.
  - Estilos para tablas, formularios, tarjetas de proyectos y footer.
